## Создание коммита
Для создания новой фиксации коммита используется команда *git commit*. Для этого в терминале с папкой репозитория пишем *git commit -m <сообщение к коммиту>*. Сообщение к коммиту писать ***ОБЯЗАТЕЛЬНО!!!***
